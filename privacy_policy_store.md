# Privacy Policy for Mobility4BW

#### Last updated: December 09, 2025

Mobility4BW is a privately developed app that provides access to open mobility data from MobiData BW. We respect your privacy and collect as little personal information as possible. This document explains what data is collected, how it is used, and which rights you have.

⸻

## 1. Data We Collect

No personal information required

Mobility4BW can be used without creating an account. We do not collect names, email addresses, or other identifying information.

### Usage Data (non-personal)

When using the app, the following information may be collected automatically:
-	Device type and operating system
-	IP address (short-term, for server communication)
-	App interactions (e.g., API requests, viewed map areas)
-	Performance and error diagnostics

These data are processed to ensure the functionality, stability, and security of the service.

### Location Data (optional)

If you grant the app access to your device location:
	Your location is used only to show nearby mobility options on the map.
	Your location is never stored on our servers and never shared.

⸻

## 2. How We Use Collected Data

Data is used exclusively to:
-	Provide the app’s features (map, mobility data, routing options)
-	Improve performance and stability
-	Fix technical issues
-	Detect misuse or errors on the API server

We do not:
-	Sell data
-	Share data with third-party advertisers
-	Build personal profiles

⸻

## 3. Third-Party Services

The app may interact with the following external data providers:
-	MobiData BW (NVBW) – Open Data for mobility (parking, car sharing, stops, etc.)
-	OpenStreetMap / MapTiler – Map tiles and geospatial data
-	Mobility4BW API Server – Optional helper service for GTFS and geocoding

These services may process technical request data such as IP addresses or timestamps to deliver their content. They do not receive personal information from us.

⸻

## 4. Data Storage

On Your Device
-	Settings, preferences, and cached mobility data are stored locally.
-	You can clear this data at any time by uninstalling the app.

On Our Server

The Mobility4BW helper API may temporarily process technical request metadata (e.g., request logs) to detect errors and ensure system stability. No user profiles are created.

⸻

## 5. Children’s Privacy

Mobility4BW is not intended for children under the age of 13. We do not knowingly collect personal data from children.

⸻

## 6. Your Rights

You may:
-	Request access to stored technical data
-	Request deletion of any information related to your device
-	Decline or revoke location permissions at any time
